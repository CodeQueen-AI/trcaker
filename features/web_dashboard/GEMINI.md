# Day 7: Web Dashboard (Streamlit)

## Today's Goal
Create a simple web dashboard using Streamlit to visualize financial data.

## Learning Focus
- Streamlit basics (widgets, layouts, charts)
- Data visualization
- Integrating with existing data files

## Fintech Concepts
- **Data Visualization**: Representing financial data graphically.
- **Interactive Dashboards**: Allowing users to explore data dynamically.
- **Real-time Insights**: Displaying up-to-date financial information.

## Features to Build

### 1. Overview Dashboard
- Display total income, expenses, and current balance.
- Show spending breakdown by category (e.g., bar chart or pie chart).
- Show income breakdown by source.

### 2. Transaction List
- Display a searchable/filterable table of all transactions.

### 3. Budget vs. Actual
- Visualize budget adherence with bar charts for each category.

## Success Criteria
✅ Can launch a Streamlit app.
✅ Displays key financial metrics (income, expenses, balance).
✅ Visualizes spending by category.
✅ Displays transactions in a table.
